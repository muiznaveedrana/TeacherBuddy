## KEY FORMULAE (Year 6)

### Perimeter
**Square**: P = 4s (or P = 4 × s)
**Rectangle**: P = 2l + 2w

### Area
**Rectangle**: A = l × w
**Square**: A = s × s (or s²)

### Other
**Distance**: d = s × t (distance = speed × time)
**Cost**: C = n × p (cost = number × price)

## USING FORMULAE

### Example: P = 4s
**If s = 7, find P**
- P = 4 × 7
- P = 28

### Example: A = l × w
**If l = 8, w = 5, find A**
- A = 8 × 5
- A = 40

### Example: C = 3n
**If n = 12, find C**
- C = 3 × 12
- C = 36

## WRITING FORMULAE

### From Words
**"Total cost is £5 per item"**
- C = 5n (where n = number of items)

**"Perimeter is 4 times the side"**
- P = 4s
