## KEY STRATEGIES (Year 6)

### Compensation
**Example**: 6,789 + 4,998
- Add 5,000: 6,789 + 5,000 = 11,789
- Adjust -2: 11,789 - 2 = 11,787

### Near-doubles
**Example**: 4,800 + 4,850
- Double 4,800: 9,600
- Add 50: 9,650

### Partitioning decimals
**Example**: 3.45 + 2.67
- Ones: 3 + 2 = 5
- Decimals: 0.45 + 0.67 = 1.12
- Total: 6.12

### Derived facts
If 48 × 25 = 1,200
Then 48 × 24 = 1,200 - 48 = 1,152
