# Year 3 Column Subtraction (No Exchange) - Mixed Layout

## Worksheet Type: MIXED LAYOUT (Fluency/Application/Reasoning)

## Structure
- **Section A: Fluency** (Q1-Q2) - Practice column subtraction skills
- **Section B: Application** (Q3-Q4) - Apply column subtraction to word problems
- **Section C: Reasoning** (Q5) - Explain or analyse column subtraction methods

## Year 3 Column Subtraction (No Exchange) Coverage

### Key Concepts
- Subtract 2-digit and 3-digit numbers using column method
- Align digits correctly by place value (ones, tens, hundreds)
- No borrowing/exchanging required (top digit always larger in each column)
- Understand why column method works for subtraction

### Number Ranges
- **Easy (Foundation)**: 2-digit - 2-digit, simple 3-digit - 2-digit
- **Average (Practice)**: 3-digit - 3-digit, mixed problems

### Example Problems (No Exchange)
- 78 - 34 = 44 (8>4, 7>3)
- 687 - 254 = 433 (each column: top > bottom)
- 896 - 573 = 323 (still no borrowing)
