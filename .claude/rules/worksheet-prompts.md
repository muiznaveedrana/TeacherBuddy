# Worksheet Prompt Rules

- Follow curriculum taxonomy for year group and topic
- Check image availability in vision catalogs before referencing
- All questions MUST have answer keys
- Max 5 questions per worksheet
- For Reception (4-5 years): visual differences must be immediately apparent
- Comparison questions MUST have 30-50%+ size difference between objects
